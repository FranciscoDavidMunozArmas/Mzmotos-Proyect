
Se requiere de NodeJS y angular para ejecutar la aplicacion

Preparacion
	Dentro de la carpeta "Frontend"
		Abre el simbolo del sistema/linea de comando/terminal
		Usa el comando "npm install"

	Dentro de la carpeta "Backend"
		Abre el simbolo del sistema/linea de comando/terminal
		Usa el comando "npm install"

Ejecucion
	Dentro de la carpeta "Frontend"
		Abre el simbolo del sistema/linea de comando/terminal
		Usa el comando "ng serve"
		
	Dentro de la carpeta "Backend"
		Abre el simbolo del sistema/linea de comando/terminal
		Usa el comando "npm start"
	
	Dentro del navegador de su preferencia ingresa a "localhost:4200"

Ingresar al sistema:
	Para el ingreso del sistema, el usuario tiene que ser validado para ello puede usar la cuentas antes creadas
